Instalar Modo Gráfico

sudo su
dpkg --configure -a
apt-get -f install
apt-get update
apt-get dist-upgrade
apt-get install --reinstall ubuntu-desktop
apt-get autoremove
apt-get clean
reboot

Comandos básicos

LS Listar directorio
ls /var/folder
CD - (change directory) cambia de directorio
cd /home/var1
Touch - Crea un archivo vacio
touch /var/file.txt

Administración

Cambiar la contraseña Root
sudo passwd root

Directorios

Borrar un directorio con su contenido adentro
rm -Rf l10n_cl_libro_compra_venta

Conexión SSH(Secure SHell)

Conectarse Remotamente
ssh root@128.0.0.1 -W 22222

Busqueda recursiva Linux & Odoo

Si queremos buscar en los ficheros
rgrep cadena_busqueda . | grep segunda_cadena_busqueda
rgrep Error . | grep .py

Comandos de Git

Instalar Git
sudo apt install git
Clonar un repositorio
git clone -b 9.0 https://github.com/dansanti/base_state_ubication
Configuración Inicial
git config --global user.name "nombre_usuario"
git config --global user.email "email_id"
Crear un repositorio
git init MyRepo
Crear un fichero Readme
gedit README
Actualizar tu repositorio local
git pull
